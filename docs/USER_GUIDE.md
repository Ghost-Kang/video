# Cascade 用户手册

> 面向短视频创作者的使用手册。内容由真实前端代码核对生成(2026-06-04),按钮文案与产品一致。

## 一句话：Cascade 是什么

Cascade 帮你把一条**爆款短视频拆开**，看懂它为什么火，然后**改成你自己的版本**，最后**一键生成你自己的短片**——从分析到出片，全程有一个 AI「导演」陪着你做。

你不用懂剪辑、不用懂脚本、不用懂运镜。你只要会刷抖音、会复制链接、会说一句「我想做支这样的视频」，剩下的交给 Cascade。

## 5 分钟快速上手

这一节告诉你：从打开 Cascade 到拿到一条可发布的短片，最短要走哪几步。

**第 1 步：进门**
- 你会看到：一个让你输入邀请码的页面。
- 你要点什么：把产品方发给你的邀请码填进去，点「进去看看」。码不对会卡在这一步进不去。

**第 2 步：粘一条你想模仿的爆款**
- 你会看到：一个让你粘抖音链接的输入框，下面还有真实案例在轮播。
- 你要点什么：把抖音视频的分享文案、App 链接或网址粘进去，点「拆解」。懒得找链接？直接点下面案例轮播里的一条也行。

**第 3 步：等它拆解（约 1 分钟）**
- 你会看到：视频封面在画面上逐幕扫描、进度点、还有已经拆出来的「钩子」。
- 你要点什么：什么都不用点，等就好。多数情况 1 分钟内出结果。

**第 4 步：看懂它为什么火**
- 你会看到：「爆点分析」卡，写着主题、为什么火、钩子/痛点/情绪/人群四个维度；往下滑还有「视频分析」逐幕拆解。
- 你要点什么：先看，心里有数。想看原视频的逐字稿，点右上角「原视频脚本」。

**第 5 步：开始做你自己的版本**
- 你会看到：分析卡上方一个深棕色大按钮。
- 你要点什么：点「在画布上做我的版本」，进入画布，它会把这条的主题/为什么火/可复制套路一起带进去。

**第 6 步：告诉导演你想做什么**
- 你会看到：一张白板，中间一张引导卡，顶部有六步流程条。
- 你要点什么：点「告诉导演你想做什么」，在底部输入框里用大白话写你的创意（比如「做一支 30 秒的宝宝辅食教程，主角是一位妈妈」），点「发送」。

**第 7 步：让导演搭好、你逐步确认和生成**
- 你会看到：导演在画布上一个个搭出节点：策划书 → 角色图 → 场景图 → 分镜 → 视频 → 合成。
- 你要点什么：选中节点 → 满意就点「✓ 确认」→ 在图片/视频节点上点「⚡ 生成」→ 最后在合成节点点「🎬 合成」拼成整片。

**第 8 步：复制去发**
- 你会看到：成片出来后的「准备发出去」卡片。
- 你要点什么：点「一键复制，去发」，把标题、话题、脚本、成片一起复制，直接去抖音粘贴发布。

## 完整流程详解

### ① 进入 & 分析

这一步是干嘛的：让 Cascade 帮你把一条爆款拆开，看懂它为什么火。

**进门。** 第一次用 Cascade，你会看到一个邀请码页面。把产品方给你的邀请码填进去，点「进去看看」。注意：码里有多余空格、或者填错了，输入框就会拦住你，进不去——这很正常，核对一下重填即可。

**粘链接。** 进来后在落地页粘一条抖音链接。你可以：
- 整段复制抖音的分享文案（带一堆字的那种）；
- 或者贴 App 里的链接、`www.douyin.com/video/...` 这样的网址。

系统会自动识别短链并展开，你不用手动处理。粘好后点输入框右边的「拆解」。

**或者直接选案例。** 不想找链接？落地页下方有真实案例在自动轮播（逐幕视频静音循环播放，也能手动翻页）。看中哪条，直接点那张案例卡，一样能进入分析。

**等待拆解。** 点完之后进入「分析中」。这时画面上会播放你刚点的那条视频的封面，逐幕扫描，配着进度点，还会把已经拆出来的「钩子」给你看。多数情况 1 分钟内出完整结果。（小提示：如果你的系统开了「减少动画」，扫描和轮播会自动关掉，只留静态封面，这是为了照顾对动画敏感的人，属于正常现象。）

**看「爆点分析」。** 拆完你会看到核心的「爆点分析」卡，里面有：
- 视频**主题**、**为什么火**；
- **英雄四维**：钩子（开头怎么抓住人）、痛点、情绪、人群；
- 以及素材利益点、主要视频元素、微创新方向、BGM 风格等。

这张卡是帮你「读懂」这条爆款的，先看明白它的套路。

**看「视频分析」。** 往下滑是「视频分析」——逐幕（一个场景一个场景）的拆解网格。每一幕都会显示：时间码、主题、描述、片段和口播、视觉内容、听觉内容，还有摄影、机位、演员、画面文字、表现形式、场景、道具、服装、光影色彩这些专业维度。想参考或改编都能从这里抄结构。

**看原脚本。** 「爆点分析」卡标题右边有「原视频脚本」按钮，点开是一个抽屉，里面是分镜脚本或完整逐字稿，可以复制（复制出来是带时间码、主题、画面、台词的格式化文本，方便你粘到笔记里）。

**进入你自己的版本。** 分析完成后，卡片上方会出现一个深棕色大按钮「在画布上做我的版本」。点它就进入画布，并把这条的主题、为什么火、可复制套路一起带进去，作为你创作的起点。

> 小贴士：时长甜蜜点是 15-90 秒，效果最好；短于 5 秒太短、长于 3 分钟拆不动。落地页下方有「时长甜蜜点」的小标签可以点开了解。

### ② 进画布 & 跟导演说需求

这一步是干嘛的：在一张白板上，用大白话告诉 AI 导演你想做什么，让它替你把活儿搭起来。

**认识画布。** 进来是一张无限大的白板，中间有一张引导卡，标题写着「这是你的创作画布」，卡里画着六步流程的流向图。顶部有一条浮窗「创作流程」六步进度条：**策划书 → 角色 → 场景 → 分镜 → 视频 → 合成**。哪步做完了是绿色✓，正在做是陶土色的呼吸圆点●，还没开始是灰色○，你一眼就知道走到哪了。

**叫出导演。** 点引导卡底部的「告诉导演你想做什么」，底部会弹起一个对话窗。引导卡上还会给你一个示范，比如「做一支 30 秒赛博朋克短片，主角是男侦探」，照着这个口吻说就行。

**说需求。** 在底部「对话创作」窗里有个输入框，提示你「描述你想创作的视频…」。你就用最自然的大白话写，比如「做一支 30 秒的宝宝辅食教程，主角是一位妈妈，温馨厨房风」。写好点「发送」（也可以 Shift+Enter 送出）。

**看导演干活。** 导演会在画布上一步步搭节点，用箭头连起来：
- 策划书（陶土色左边框）；
- 角色图、场景图、分镜表（橙色图片节点）；
- 视频镜头（琥珀色）；
- 合成整片（棕色）。

**收起 / 再叫出对话。** 想看清画布，点对话窗右上角的「收起对话」把它折叠；折叠后右下角会有一个陶土色圆形浮球「问导演」，随时点它再展开。

### ③ 六步创作

这一步是干嘛的：跟着顶部那条六步进度条，把一支短片从「文字方案」一路做到「成片」。

六步分别是：
1. **策划书**：导演写的文字方案（为什么火 + 改成你的版本）。
2. **角色**：定主角形象。
3. **场景**：定拍摄场景。
4. **分镜**：定每个镜头的画面。
5. **视频**：逐镜生成短片。
6. **合成**：把所有镜头拼成整片。

你不用记顺序——导演会按这个顺序搭，进度条会实时点亮。你的任务是在每一步「看一眼、满意就确认、然后生成」。

### ④ 逐节点生成与审核

这一步是干嘛的：对导演搭好的每个节点，由你来确认内容、点生成、把它真正做出来。

**选中节点看操作条。** 在画布上点任何一个节点，它**上方会浮出一条操作条**，按节点类型显示不同按钮：
- 策划书节点：「✓ 确认」和「↻ 重生」；
- 生成中的图片/视频节点：「✕ 取消」；
- 已确认的图片/视频节点：「⚡ 生成」；合成节点：「🎬 合成」。

**确认策划书。** 策划书是创作第一步。看完导演写的方案，满意就点「✓ 确认」，这会解锁下游节点的创建。想改？点「↻ 重生」会弹出一个输入框问你「怎么改？（可留空=自动）」，写下你的意见（留空就让导演自动改）再提交，导演会按你的反馈重写，旧版自动存进版本历史。

**生成图片。** 选中图片节点，右侧详情面板里可以：选图片生成的服务商（Apimart 或 Google Gemini）、写或改 Prompt（生成指令，比如「一位妈妈在明亮厨房给宝宝做辅食，温馨」），然后点「生成」。生成中会有呼吸动画和「生成中」状态。图片通常较快。

**生成视频。** 选中视频节点，详情面板里可以设：时长（4-15 秒）、分辨率（480p / 720p / 1080p）、是否配音（开/关），改好 Prompt 点「生成」。视频要久一些，会显示「生成视频中…（约几分钟）」，耐心等。

**润色 Prompt（可选）。** 图片/视频 Prompt 框下方有「润色」。点开后写下你想怎么改，点「提交」，系统会自动帮你优化这段 Prompt。不想改就直接生成，用你写的这版。

**合成整片。** 视频都做好后，选中合成节点点「🎬 合成」（详情面板里也可点「合成」）。合成节点没有 Prompt 可编辑，它会自动把前面的视频拼起来，显示「合成中...」，等它完成。

**生成中可取消。** 任何生成/合成进行中，操作条上的按钮会变成「✕ 取消」，你可以随时中止，之后再重新启动。

**看状态芯片。** 每个节点左下角有个彩色小徽章，一眼看懂它在干嘛（按优先级）：
- 「⚠ 待重生」（琥珀告警）：上游变了，这个产物过时了，要重做；
- 「生成中」（琥珀呼吸）：AI 正在处理；
- 「✕ 失败重试」/「✕ 超时重试」（红色）：出错了，可以重来；
- 「✓ 已确认」/「已生成」（绿色）：通过了或做好了；
- 「● 待确认」（琥珀）：内容好了但还没经你审核。

**详情面板的审核切换。** 右侧详情面板顶部可以在「审核中」和「已确认」之间切换这个节点的状态。

**参考图（仅图片/视频节点）。** 如果有上游节点的产物（图或视频），会作为「参考图」显示在详情面板里，用来指导这个节点生成。每张图右上角有「✕」可删除；多张图时可用「›」「‹」调整先后顺序。

**审核闸门。** 有时导演想自动一口气生成一批内容，这时会弹出一张卡，标题「待你确认是否生成」，列出它要生成什么。你可以点「确认生成」放行（会消耗配额），或点「先不生成」拦住它。这是给你一道最后的人工把关，免得不小心生成一堆。注意：**你自己**显式点「⚡ 生成」或「🎬 合成」时不会弹这张卡，会直接执行——审核闸门只拦导演的自动生成。

### ⑤ 回溯改上游

这一步是干嘛的：做到一半发现前面某步想改，回去改它，并让后面依赖它的部分跟着更新。

**重生上游。** 在想改的节点上点「↻ 重生」：
- 如果是策划书，会弹输入框让你写改稿意见（可留空自动改）；
- 如果是图片/视频，会先自动把当前产物存档，再重做。

重生完成后，所有依赖它的下游节点会自动标上「⚠ 待重生」（琥珀警告，按钮也变琥珀色强调），提醒你这些后续产物基于旧上游、已经过时，需要重新生成。这条「级联更新链」是为了保证你最终的成片是前后一致的。

**版本历史 & 对比。** 右侧详情面板底部是「版本历史」，列出这个节点的所有历史版本（v1、v2……），每条显示版本号、改稿原因、时间。点选其中一条，可以把当前版和这个旧版的产物放一起对比。

**一键回滚。** 在版本历史里选中某个旧版，如果当前不在生成中，可以点「↩ 回滚到此版」，一键恢复到那一版。放心：回滚不是删除——你当前这版会被自动存档，以后随时还能再切回来。回滚后，下游同样会被标「⚠ 待重生」。

> 记住两点：① 版本历史**只记录「重生」**，中间你手动改的 Prompt 不会单独存版本；② 回滚和重生都会让下游标脏，这是正常的链条更新，不是出错。

### ⑥ 发布

这一步是干嘛的：把做好的成片连同标题、话题、脚本一起打包，复制出去直接发抖音。

成片生成后，你会看到一张「准备发出去」卡片：
- 点「复制标题」：把建议标题复制到剪贴板；
- 点「复制话题」：把话题标签（带 # 的那一串）复制到剪贴板；
- 点「复制脚本」：把改好的脚本复制到剪贴板；
- 点「一键复制，去发」：把**完整发布包**（标题 + 话题 + 脚本 + 镜头图 + 成片链接）一次性复制到剪贴板，直接去抖音粘贴。

每次复制后，界面顶部会弹一个成功/失败提示，3 秒后自动消失。

> 小提示：发布包里的「镜头图」在草稿图还没生成完时，会显示「(草稿图生成后自动填入)」的占位；「成片」在合成完成前会显示「(合成整片后自动填入)」。等它们做完，发布包会自动补上，你再复制就是完整的了。

## 画布操作速查

这一节是干嘛的：把画布上常用的「移动、排版、连线、导航、复用」操作集中告诉你。

**缩放与适应视图。** 画布左下角有一组缩放控件，含「放大」「缩小」「重置视图」，用来放大缩小，或一键把全部内容缩到屏幕里看全景。

**自动排版。** 画布右上角有「自动排版」按钮，点一下自动把节点理整齐，看着清爽。（章节分组里的镜头保持相对位置不动。）

**手动移动节点。** 直接拖拽节点就能挪位置，你摆的位置会被记住。

**连线。** 从一个节点的源点拖到另一个节点的目标点，可以新增一条连线（表示依赖关系）。想删连线，把鼠标移上去，点中间出现的「✕」即可。

**章节分组。** 画布右侧会按叙事结构（开场、发展、高潮、结尾）自动把视频镜头分成四个浅色框组，框内镜头相对排列，不参与全局自动排版。

**小地图。** 当画布上的节点**达到 8 个或以上**时，右下角会出现小地图（缩略导航），不同颜色代表不同节点类型（陶土=脚本、橙色=图片、琥珀=视频、棕色=合成），可以在上面拖拽平移或拉框缩放快速定位。节点少于 8 个时不显示小地图，这是为了避免它贴在节点上挡住你，属于正常设计。

**历史会话（左侧栏）。**
- 点顶栏「+」新建一个创意会话；
- 点任意历史会话卡片，切换到那个方案继续看或继续做；
- 鼠标悬停在卡片上，点铅笔图标可「重命名」（按 Enter 确认、Esc 取消）；
- 悬停后点垃圾桶图标可「删除」该会话（注意：只有会话多于一个时才出现垃圾桶，至少给你留一个）；
- 有空会话时会出现「清理空会话」，点它再确认一次，会清掉那些没拆解完又没命名的空会话。

**素材复用（右侧「你之前用过的」面板）。** 没选中节点时，右侧会显示「你之前用过的」，分「你的角色」和「你的场景」两类，列出你之前用过的素材：
- 左上角可切「按使用次数」或「按时间」排序；
- 点任意角色/场景卡片即可快速复用，省得重做；
- 被复用过至少一次的素材，右上角会标「已用 X 次」。

## 名词小词典

这一节是干嘛的：把画布里那些「黑话」翻译成你能秒懂的大白话。

| 名词 | 大白话 |
|---|---|
| **邀请码** | 内测期间进门用的「门票」，由产品方发给你；填错或带空格就进不去。 |
| **爆点分析** | 拆完后那张核心卡，告诉你这条视频的主题、为什么火、钩子/痛点/情绪/人群，帮你读懂它的套路。 |
| **视频分析** | 一幕一幕的详细拆解表，时间码、画面、台词、运镜、服装、光影都有，方便你照着改。 |
| **钩子** | 视频开头那几秒里「抓住人、让人不划走」的招数。 |
| **可复制套路** | 从这条爆款里总结出来的结构模板，你可以套这个骨架换成自己的主题。 |
| **导演** | 帮你干活的 AI。你说创意，它在画布上一步步搭出策划书、图、视频。 |
| **节点** | 画布上的一张张卡片，代表创作的一个环节：策划书、图片、视频、合成。 |
| **策划书** | 导演写的文字方案（剧本、分镜、清单），创作的第一步。 |
| **连线 / 箭头** | 节点之间的连接，表示「谁是谁的参考、谁依赖谁」。 |
| **锚点 / 素材** | 你用过的角色、场景，会存在右侧「你之前用过的」里，能直接复用。 |
| **Prompt（提示词）** | 你对 AI 的描述指令，用大白话说清要生成什么样的图或视频。 |
| **Provider** | 生成图片用的服务商，可在 Apimart 和 Google Gemini 之间选。 |
| **生成** | 让 AI 真正把图或视频做出来。 |
| **合成** | 把做好的多条视频拼成一个完整短片。 |
| **重生** | 改了条件或上游后，重新做一遍这个节点；旧版会自动存档，能对比、能回滚。 |
| **回滚** | 退回到之前某个版本；当前版本会自动存档，随时能再切回来。 |
| **审核（审核闸门）** | 导演想自动生成一批东西时弹出的确认卡，让你点「确认生成」或「先不生成」把关。 |
| **状态芯片** | 节点上的彩色小徽章，一眼看懂它现在是待确认、生成中、已完成还是要重做。 |
| **待重生** | 上游改了，这个产物过时了，得重新生成。 |
| **润色** | 让系统帮你把 Prompt 写得更好——你说改的方向，它来优化。 |
| **发布包** | 把成片连同标题、话题、脚本、镜头图整理成一包，一键复制去抖音发。 |
| **草稿图** | AI 按画面描述画的每一幕参考图，用来预览成片效果。 |
| **成片** | 把所有镜头合成出来的完整短视频。 |

## 常见问题 FAQ

这一节是干嘛的：把你可能会卡住的地方，提前用问答说清楚。

**Q：我填了邀请码却进不去？**
A：检查邀请码本身——多余的空格、填错的码都会在输入框被拦住。核对后重新填，点「进去看看」。

**Q：链接要怎么粘？非得是纯网址吗？**
A：不用。整段复制的抖音分享文案、App 链接、`www.douyin.com/video/...` 网址都行，系统会自动识别短链并展开。或者干脆点落地页下面的案例卡，跳过粘链接。

**Q：什么样的视频最适合拆？**
A：时长 15-90 秒的最好。短于 5 秒太短、长于 3 分钟拆不了。落地页下方的「时长甜蜜点」标签里有详细说明。

**Q：分析中为什么没有扫描动画，只有一张静态封面？**
A：如果你的系统开了「减少动画」，扫描线和逐幕轮播会自动关闭，只留静态封面。这是照顾对动画敏感的人的无障碍设计，不是出 bug。

**Q：点了「⚡ 生成」怎么半天没动静？**
A：生成是**异步**的——点完状态立刻变「生成中」，但真正做完要时间。图片通常较快，视频通常要几分钟。等它完成才能进下一步或改 Prompt；等不及可以点「✕ 取消」。

**Q：策划书的「↻ 重生」怎么和图片的「生成」不一样？**
A：图片/视频的生成是一键出活。策划书的「重生」会先弹一个输入框问你「怎么改？（可留空=自动）」，写下意见（或留空让导演自动改）提交后，它才按你的反馈重写，并把旧版存档。

**Q：我改了一个节点，后面好几个节点都变「⚠ 待重生」了，是出错了吗？**
A：不是。这是「级联更新」——你重生了上游，所有依赖它的下游会自动标「待重生」，提醒你它们基于旧上游、需要重新生成。这样最终成片才前后一致。

**Q：回滚到旧版，是不是就把现在这版弄丢了？**
A：不会。回滚不是删除——你当前这版会自动存档成新版本，以后随时还能切回来。版本历史里能看到所有版本。

**Q：为什么我手动点生成不弹审核卡，导演生成时却弹？**
A：审核闸门只拦**导演的自动生成**，给你最后一道把关、免得意外消耗配额。你自己显式点「⚡ 生成」或「🎬 合成」，等于你已经决定了，会直接执行，不再弹卡。

**Q：怎么删一个连线？**
A：把鼠标移到连线上，点中间出现的「✕」即可删除。要删某张参考图，是在节点详情面板里那张图的右上角点「✕」，不在操作条上。

**Q：为什么我的画布没有小地图？**
A：节点不到 8 个时不显示小地图，免得它贴在节点上挡住你。等节点多了、画布溢出了，小地图会自动出现。

**Q：历史会话里的垃圾桶按钮怎么不见了？**
A：只有当你有多于一个会话时才会出现垃圾桶，系统至少给你留一个会话不让删。

**Q：发布包里「镜头图」「成片」怎么是占位文字？**
A：草稿图还没生成完时，镜头图会显示「(草稿图生成后自动填入)」；整片还没合成时，成片会显示「(合成整片后自动填入)」。等它们做完会自动补上，到时再复制就是完整发布包。

**Q：复制成功了吗？**
A：每次复制后界面顶部会弹一个成功或失败的提示，3 秒后自动消失。看到成功提示就说明已经在你的剪贴板里了，直接去抖音粘贴即可。
