## 1. 2026 中文 AIGC 视频赛道：谁在赢个人创作者

### 1.1 即梦 AI（字节跳动 / 剪映系）

**【数据支持·训练】**
- **定位**：字节内嵌「文/图生图 + 图生视频」工具，与剪映/抖音强绑定
- **目标用户**：抖音创作者（特别是抖音电商商家）、剪映用户基本盘
- **2026 Q1 定价**：基础免费（每日积分），会员 ~¥69-79/月（高清/优先队列/视频时长解锁）；按次购买积分包 ~¥30/600 积分
- **强项**：抖音生态分发位（首页有入口）、character reference（"分身"功能）、与剪映无缝跳转
- **不解决**：
  - ❌ 不告诉你**该拍什么**（无热点雷达/选题推荐）
  - ❌ 不分析爆款（只生成，不学习）
  - ❌ 没有「跨视频角色一致性」的资产库形态（character 是隐藏字段）
  - ❌ 不做"复刻"——把别人的爆款转成你的版本

**对 Cascade 含义**：即梦是**生成层最强对手**，但它把"画布 + 学习 + 复刻"完全空着。Cascade 不在生成质量上正面竞争（生成接 Apimart/Seedance/Kling），而是在**生成之上的工作流**。

### 1.2 可灵 AI（快手）

**【数据支持·训练】**
- **定位**：纯视频生成模型 + 简单产品壳
- **目标用户**：专业创作者 / 早期采用者 / 海外开发者（API 客户占比大）
- **2026 Q1 定价**：免费每日少量积分；黄金会员 ¥66/月；铂金 ¥266/月；钻石 ¥666/月——分别对应每月可生成视频数和高级模型权限
- **强项**：模型质量公认 Top tier（中文圈中和 Sora 2 / 即梦三足鼎立）、API 开放、海外用户群（独有的优势）
- **不解决**：
  - ❌ 产品壳粗糙（基本是 prompt + 参数）
  - ❌ 无热点/无选题/无 URL 分析
  - ❌ 角色一致性靠用户自管 reference 图
  - ❌ 中文短视频创作生态外挂的痛感重（导出 → 剪映 → 抖音的链路断裂）

**对 Cascade 含义**：可灵的 API 是**Cascade 视频片段生成的可选后端之一**（与 Seedance 并列）。可灵的产品形态恰恰**反证 Cascade 的差异化命题**——纯模型公司做不了"工作流"。

### 1.3 Sora 2（OpenAI，2025 末发布）

**【数据支持·训练】**
- **定位**：通用视频生成 + character lock + 物理仿真
- **目标用户**：海外开发者 / 高端创作者 / 企业
- **2026 Q1 定价**：ChatGPT Plus ¥150 等值（受限额度），Pro ¥1500 等值（高清+长时长）
- **强项**：物理仿真和镜头连贯性领先半代；character lock 体验完整
- **不解决**：
  - ❌ **中国区不可直接访问**（合规壁垒，需要中转）
  - ❌ 中文 prompt 优化弱（训练偏英文+西方审美）
  - ❌ 不懂抖音/小红书审美（一眼"AI 海外感"）
  - ❌ 无中文热点输入

**对 Cascade 含义**：**Sora 2 不是 Cascade 的直接对手**——它服务的是另一拨人。但 Cascade 可以在内容上学习 Sora 2 的产品体验（特别是 character lock 的可视化做法）。

### 1.4 剪映 AI（字节 / CapCut 国内版）

**【数据支持·训练】**
- **定位**：剪辑工具 + AI 功能叠加（AI 字幕、AI 配音、AI 抠图、AI 文生图、AI 图生视频）
- **目标用户**：**整个抖音/小红书创作者基本盘**（亿级 MAU）
- **2026 Q1 定价**：基础免费；剪映 VIP ¥38/月、¥218/年；AI 创作会员 ¥58/月（含即梦积分）
- **强项**：**用户基本盘最大**（创作者已经在用）、与抖音无缝、移动端体验最优
- **不解决**：
  - ❌ "**剪辑工具叠 AI 功能**"≠"**AI 优先工作流**"——AI 是辅助，不是主线
  - ❌ 不做 URL 学习/复刻
  - ❌ 不做热点选题
  - ❌ Web/桌面端弱（移动优先）

**对 Cascade 含义**：剪映是**目标用户日常使用的产品**——Cascade 的**发布包必须无缝接剪映**（不能让用户离开他们的肌肉记忆链路）。**不要与剪映竞争剪辑层**。

### 1.5 海螺 AI（MiniMax）

**【数据支持·训练】**
- **定位**：通用对话 + 多模态（文/图/音/视频）助手
- **目标用户**：通用 C 端 + 创作者尝鲜（视频是其多模态之一，不是产品核心）
- **2026 Q1 定价**：免费每日额度；会员 ¥49/月（提升所有模态额度）；视频生成单独按次/按积分
- **强项**：TTS 质量（海螺音色是行业标杆）、多模态融合
- **不解决**：
  - ❌ 视频不是主线（产品广度优于深度）
  - ❌ 无创作者工作流（无画布/无热点/无 URL 学习）

**对 Cascade 含义**：海螺的 **TTS 是 Cascade 的可选后端**（与火山豆包并列）。竞争层不正面。

### 1.6 其他相关玩家

**【数据支持·训练】**
- **腾讯混元视频**：偏 To B，C 端创作者渗透弱；¥39-99/月会员
- **智谱清影**：技术派，产品壳弱
- **通义万相**（阿里）：电商场景强（淘宝商品视频自动化），不直接对个人创作者
- **Vidu**（生数）：3D-aware 视频，专业向，C 端弱
- **Pika / Runway**：海外，中文创作者渗透 < 5%

### 1.7 工具数据类玩家（互补，非竞争）

**【数据支持·训练】**
- **新榜**：B2B 数据订阅 ¥3000-30000/年起，个人创作者**几乎不直接付费**
- **飞瓜数据**：抖音/小红书数据分析，个人版 ¥98/月起，付费用户主要是直播带货从业者
- **卡思数据**：达人分析，B2B 为主
- **蝉妈妈**：电商数据，与创作者工具距离远

**对 Cascade 含义**：这些工具的**数据**是 Cascade 的输入（hotspot 整合中），但它们的**产品形态**（dashboards + Excel 导出）和 Cascade（画布 + 创作）天然不同——它们看完数据后**用户还要去别的工具创作**。Cascade 把"看 + 做"合二为一是**真实的产品差异化**。

## 2. "爆款复刻"工作流真空：是否有人做了"粘贴 URL → 你的版本"闭环

### 2.1 现状判断

**【数据支持·训练 + 我推测】**

**没有任何一个面向 C 端创作者的产品**把"粘贴抖音/小红书 URL → 自动拆解 → 赛道改写 → 生成你的版本"做成主线 UX。下面是邻近赛道：

| 产品 | 做了什么 | 没做什么 |
|---|---|---|
| **Topview.ai** / **Captions.ai**（海外） | URL → 字幕提取 / 重剪 | 不做"赛道改写"；不做"画布 + 锚点复用"；不服务中文生态 |
| **Opus Clip** / **Vizard** | 长视频切短（克隆 hook） | 不做"输入爆款 → 输出全新视频" |
| **新榜 / 飞瓜** "拆解爆款"栏目 | LLM 文本拆解 hook/节奏（无生成） | 数据展示，不生成 |
| **抖音热点宝**（官方） | 看热点话题 | 不创作 |
| **小红书 蒲公英 / 灵感平台**（官方） | 看选题灵感 | 不创作 |
| **剪映"模板"** | 用别人的模板套自己的素材 | **这是最近的形态**，但模板是**剪辑层模板**——节奏/转场固定，文案/角色/场景需要用户自带；不分析"为什么火" |
| **抖音"同款"按钮** | 用同样 BGM/特效 | 用户素材自备，不生成内容 |

### 2.2 关键洞察

**【我推测·高置信度】**

剪映"模板"功能是**用户对"复刻"概念已经被教育**的强证据——亿级用户每天在套模板。但剪映模板有三个局限：
1. **素材用户自备**——不是从零生成，宝妈还是要自己拍/找素材
2. **公式是黑盒**——不告诉你"为什么这条火"
3. **不跨平台/跨赛道改写**——抖音模板就是抖音模板

**Cascade 的产品定位翻译成用户语言**：「**剪映模板 + 自动生成素材 + 解释为什么火 + 改写到你的赛道**」。这个表达方式应该在 landing page 测试。

### 2.3 为什么大厂不做

**【我推测】**

字节/快手/腾讯都**有能力**做"URL → 复刻"但**不做**，原因可能是：
1. **版权风险**——大厂法务红线（创作者可以借鉴，平台不能"批量改写"）
2. **平台中立**——抖音不能官方推广"复刻"功能，否则伤害原创者生态
3. **变现路径不清**——大厂的 KPI 是 DAU/时长，不是创作者 LTV

**对 Cascade 含义**：这是**结构性的空窗期**——大厂不会做，小厂没胆做（toprador 的视频分析能力是 Cascade 的不可替代护城河）。但**版权风险是产品级风险**（PRODUCT_VISION §8 反共识假设 #3 已识别）——Phase 0 期间必须有法务咨询，至少明确："分析 + 公式提取 + 用你的资产重做"是法律安全的，"像素级复制"不是。

## 3. 30岁宝妈辅食创作者的当前替代方案（用户痛点基线）

### 3.1 当前真实工作流

**【数据支持·公开报告 + 我推测】**（基于公开创作者访谈和小红书/B站的"我的创作流程"视频内容）

宝妈辅食赛道一条普通视频的诞生流程（**当前基线，没有 Cascade 的世界**）：

```
1. 选题阶段 (30-60 分钟)
   - 刷抖音/小红书 "对标账号" 看最近爆款
   - 截图存 "灵感库"（手机相册一个文件夹）
   - 看 飞瓜/灰豚 的"昨日爆款" 列表（部分付费用户）
   - 凭感觉抄一个：标题 hook + 选材 + 拍摄角度

2. 文案/脚本 (30-60 分钟)
   - 看 3-5 条同主题视频，手抄标题/前 3 秒台词
   - ChatGPT/豆包 改写文案（25 年起这已普及）
   - 自己写"宝妈口吻"（AI 写出来太书面，用户主动改）

3. 拍摄 (60-120 分钟)
   - 厨房支手机，自己拍
   - 同一个食材拍 5-10 个角度
   - 出镜的话：化妆 30 分钟 + 录制 60 分钟 + 多次 NG

4. 剪辑 (60-120 分钟)
   - 剪映：手动卡点 + 套模板 + AI 字幕 + AI 配音（部分）
   - 调色、加贴纸、加 BGM
   - 反复预览

5. 发布 (15 分钟)
   - 写标题/标签（参考爆款）
   - 选封面
   - 发抖音 + 同步小红书 + 同步视频号

总时长：3-7 小时/条；一周 2-3 条更新；月产 8-12 条
```

### 3.2 痛点排序（每一条对应 Cascade 的功能位）

**【我推测·需 Phase 0 访谈验证】**

| 痛点 | 频次 | Cascade 解决方案 | Phase 1 是否做 |
|---|---|---|---|
| **不知道拍什么** | 每天 1 次 | URL 输入 + 系统精选 5 条热点 | ✅ P1-1 |
| **抄爆款不知道为啥火** | 每条视频 | 浅分析 whyItHit + howToAdapt | ✅ P1-2 |
| **拍摄太累（特别是出镜）** | 每条 | 不解决拍摄；但**生成式镜头**可以替代部分实拍 | ⚠️ Phase 2 |
| **文案没灵感** | 每条 | 赛道改写脚本 | ✅ P1-3 |
| **同一角色/家人多次出镜耗时** | 持续 | 锚点跨 run 复用 | ✅ P1-6 |
| **跟不上 KOL 更新频率** | 持续 | 时间压缩（3 小时 → 30 分钟） | ✅ 总目标 |

### 3.3 不应低估的"现有解很好"

**【我推测·必须验证】**

- 剪映的"AI 文案 + AI 配音 + AI 字幕"在 2025 年已经把单条剪辑从 2 小时压到 40 分钟。**目标用户的"AI 红利"已经吃了一半**——Cascade 必须解决**剪辑之外**的（选题 + 学习 + 改写）。
- "拍摄"环节用户不希望被取代——很多宝妈认为**真人出镜是信任的来源**。**Cascade 不要走"完全 AI 化"的死胡同**，要走"AI 处理工业部分，人保留情感部分"。

**对 Phase 1 决策含义**：**P1 不做完整 60s 自动成片是对的**（PHASED_PLAN §4.3 已经砍掉）。**真正的甜点是脚本 + 关键镜头草稿 + 发布包**——让用户拿这些素材回到剪映/抖音去补真人镜头。

## 4. 付费意愿基准：¥39/月假设的现实检验

### 4.1 中文创作者付费基线

**【数据支持·训练 + 公开报告】**

| 工具类别 | 主流价位 | 付费率（粗略） |
|---|---:|---:|
| 剪映 VIP | ¥38/月 / ¥218/年 | C 端创作者付费率 ~10-15% |
| 即梦/可灵 会员 | ¥49-79/月 | 个人创作者付费率 < 5%（多数走免费每日额度） |
| 飞瓜/灰豚 个人版 | ¥98-198/月 | 主要是带货从业者付费，纯内容创作者 < 3% |
| ChatGPT Plus / 国内同等 | ¥120-200/月 | 早期采用者，渗透率 < 5% |
| 千图网/包图网 素材会员 | ¥300-600/年 | C 端 ~10-15% |
| 新榜个人版 | ¥600+/年 | 极小众，主要是 MCN 运营 |

### 4.2 关键数据点对 ¥39 假设的影响

**【数据支持 + 我推测】**

1. **¥39/月在剪映 VIP（¥38）的锚定区**——这是**心理上"AI 工具单一付费"的天花板**。**结论：¥39/月定位正确，但意味着 Cascade 必须证明自己"和剪映一样必要"**。
2. **目标用户「30 岁宝妈」对单 SaaS 工具的付费意愿现实是 ¥0-50/月**——超过 ¥99 需要明确变现路径（比如带货流量翻倍）。
3. **关键反直觉**：付费动机**不是「省时间」**（创作是热爱），是**「视频火了」**——**Cascade 的留存和付费转化与"爆款命中率"强绑定**，而不是"工具好用"。
4. **大量用户用「批发购买积分包」而不是订阅**——这是中文 C 端 AI 工具的特殊形态。即梦/可灵都有按次积分包（¥30/600 积分）。**Cascade 应当 day-1 同时支持订阅 + 积分包**（PHASED_PLAN Phase 2 已预留 schema）。

### 4.3 ¥39 假设的隐藏陷阱

**【我推测·高置信度，建议 Phase 0 验证】**

PRODUCT_VISION §2.1 写 "¥39/月可接受"——这个**「接受」语义模糊**。区分三种语义：

| 语义 | 含义 | Phase 0 验证手段 |
|---|---|---|
| 「问起来可以接受」 | 用户在访谈中说"39 还行" | ❌ 这是社交答案，不准 |
| 「看到价格后试用」 | landing page 显示 ¥39，用户填邮箱 | ✅ 中等强度信号 |
| 「拿出信用卡」 | 用户真的扣款且 30 天后续费 | ✅ 唯一可信信号 |

**强烈建议**：Phase 1 Gate 中 "至少 2 人说'它帮我把爆款变成我的内容'" 应该**升级为** "至少 2 人**预付** ¥39 且 30 天后用过 ≥ 3 次"。否则 Phase 2 的 "5 人付费" gate 是数字幻觉。

## 5. 抖音 / 小红书 / 微信 三平台创作者获取渠道

### 5.1 创作者发现新工具的真实路径

**【数据支持·训练 + 公开报告】**（基于 2024-2025 多个创作者调研报告，方向稳定）

按转化效率排序：

| 渠道 | 转化效率 | 触达成本 | 适合阶段 |
|---|---|---|---|
| **小红书 KOL/KOC 测评帖**（中腰部，1-10w 粉） | ⭐⭐⭐⭐⭐ | 单帖 ¥500-5000 | Phase 1-2 |
| **抖音"创作者教学"账号合集** | ⭐⭐⭐⭐ | CPM 较高，需要内容能力 | Phase 2 |
| **微信视频号 + 公众号长文测评** | ⭐⭐⭐⭐ | 长尾，作者私交起步 | Phase 1 |
| **垂类微信群 / QQ 群**（"宝妈创作者交流"等） | ⭐⭐⭐⭐ | 几乎免费，需要社群运营人 | Phase 1 |
| **B 站工具测评 up 主** | ⭐⭐⭐ | 中长期 SEO 价值 | Phase 2 |
| **抖音直播 + 私域沉淀** | ⭐⭐⭐ | 高强度运营 | Phase 2 |
| **百度搜索 / SEO** | ⭐⭐ | 长尾，AI 工具关键词竞争激烈 | Phase 3 |
| **应用商店 ASO** | ⭐⭐ | 中文 ASO 不像海外有效；且 Cascade 是 Web 不是 App | 不适用 |
| **知乎专栏** | ⭐⭐ | 转化率低但人群精准（高净值） | Phase 2 |

### 5.2 三平台对工具发现的不同心智

**【我推测·中等置信度】**

| 平台 | 用户心智 | 工具帖类型 | Cascade 适配 |
|---|---|---|---|
| **小红书** | "实测/避雷/真心推荐" | 图文测评 + 流程截图 | ✅ 最适配——目标用户基本盘在这 |
| **抖音** | "看效果，1 分钟内决定" | 工具使用 before/after 短视频 | ✅ 适配——展示 "粘 URL → 30 分钟出片" 全过程 |
| **微信** | "朋友/同行推荐，深度文" | 公众号长文 + 私域成交 | ✅ 适配 Pro 转化——长文讲解"为什么爆款公式可学习" |

### 5.3 创始人个人品牌的杠杆

**【数据支持·公开报告 + 我推测】**

PRODUCT_VISION §1.3 提到 "推特/小红书 KOL 圈、创始人个人品牌驱动获客"。**Twitter/X 在中文创作者圈渗透率 < 3%**——是中文产品的**错误获客渠道**。建议修正：

- **推特/X → 砍掉**（不是中文创作者聚集地，除非目标是海外 AI 圈背书）
- **小红书 → 加强**（创始人个人账号 day-1 起号，每周 2 条"我是怎么用 AI 做爆款"）
- **微信视频号 + 公众号 → 加**（更适合"创业故事 + 工具解决方案"长内容）
- **即刻 → 可选**（创业者/产品经理圈，对 Cascade 这种产品有天然好感，可做 BD 起点但不是用户起点）

**Phase 1 GTM 建议**：创始人自己用 Cascade 做 10 条宝妈辅食视频发到小红书 + 抖音，**用产出本身做内容营销**——这同时验证产品 + 起号 + 形成 case study。

## 6. 三大差异化 Angle（排序 + Phase 1 决策含义）

基于 §1-§5 的市场真空和用户痛点，给三个**Phase 1 必须押注的**差异化 angle，按"市场容易感知 × 我方易守"排序：

### 6.1 第一 Angle：「**复刻**」——粘 URL → 你的版本（市场感知度最高）

**为什么排第一**：
- §2 显示这是**结构性空窗**（大厂不敢做、小厂没能力）
- §3 显示宝妈**已经在手动做这件事**（截图 + 抄 hook + 改文案），Cascade 只是"自动化她正在做的事"——**无需教育市场**
- §1.2 / §1.4 显示**生成层公司都没做这件事**——可灵/即梦的盈利在 API 和会员，不会下沉到工作流
- 一句话定位（PHASED_PLAN §0）已经收敛为这个，**与 Phase 1 决策完全对齐**

**Phase 1 决策含义**：
- ✅ P1-1（URL 输入框）必须做首位
- ✅ 浅分析卡片 whyItHit / howToAdapt 必须是产品的**主视觉**，而不是隐藏功能
- ⚠️ landing page 文案必须用「**把别人的爆款变成你的版本**」这个表述，不要用「AI 视频生成」（与即梦/可灵无法区分）

### 6.2 第二 Angle：「**赛道改写**」——同一爆款公式 + 你的赛道资产

**为什么排第二**：
- §3.2 显示用户的**真实痛点是「我没法照搬，因为我是辅食赛道，对标博主是穿搭赛道」**——这是即梦/可灵彻底空白的层
- §1.4 剪映模板的局限恰恰证明：**用户已经接受"复刻"概念，但模板的「赛道适配性」缺失**——Cascade 的 LLM 改写填的就是这个洞
- **护城河更深**：URL 输入是产品形态，**赛道改写质量**才是真的需要数据 + 模型 + 时间积累的能力

**Phase 1 决策含义**：
- ✅ P1-3（赛道改写 prompt + 脚本输出）是 Phase 1 真正的**产品价值核心**，比生成图还重要
- ⚠️ Phase 1 第一批 10 个用户最好**集中在 2-3 个赛道**（如：宝妈辅食 + 育儿日常 + 家庭厨房）——这样改写 prompt 可以针对性优化
- 🚨 **Gate H3（改写质量足以让用户继续）必须严格执行**——如果改写 prompt 出来的脚本用户 50% 重写，这个 angle 就没成立

### 6.3 第三 Angle：「**锚点资产沉淀**」——用得越久越懂你

**为什么排第三**（不是不重要，而是 Phase 1 不容易被用户感知）：
- §1.1-§1.5 全部对手都把"character reference"做成隐藏字段，**用户对"角色锚点"概念完全没建立心智**
- 用 1 次的用户感受不到这个 angle 的价值（必须用 3+ 次才能感受到"我的辅食角色小张"）
- §3 用户当前**没有这个痛点意识**（她们不知道"建立可复用角色"是更好的工作流）
- 但 PHASED_PLAN §7 H8（系统从重复使用中变聪明）是**Buffett 护城河**的核心——Phase 3 商业化的命门

**Phase 1 决策含义**：
- ✅ P1-6（锚点跨 run 复用最小版）必须做，但**不要主推**——产品介绍页可以不提"锚点"概念
- ⚠️ 让锚点**自然出现**——用户第二次创作时系统说"上次你用的小张妈妈，要不要继续用？"
- 🚨 **Gate H4（锚点跨 run 复用感知）是 Phase 2 决策点**——Phase 1 验证它"可以发生"，Phase 2 验证它"用户重视"

### 6.4 不推荐的 Angle（避坑）

| Angle | 为什么不要 |
|---|---|
| "AI 视频更便宜" | 即梦/可灵免费额度已经覆盖宝妈每周 2-3 条更新 |
| "AI 视频更快" | 剪映 + AI 字幕/配音已经把效率拉到 40 分钟 |
| "AI 视频质量更好" | Cascade 不做模型层，正面比拼必败 |
| "全自动 60s 成片" | §3 已说明用户**不希望**完全 AI 化（信任来自真人） |
| "多模式画布" | PHASED_PLAN 已砍，Phase 1 应专注「一个模式做好」 |
| "热点数据中台" | 新榜/飞瓜的位置——B2B 工具的盈利模型，C 端不会为"看数据"付费 |

## 8. 「This changes Phase 1 strategy because…」5 条结论

每一条对应 PHASED_PLAN Phase 1 的具体决策修订建议：

1. **GTM 渠道修正：把"推特/X"砍掉，all-in 小红书 + 微信视频号 + 创始人自身做 10 条 case study**（§5.3）。当前 PRODUCT_VISION §1.3 的 "推特/小红书 KOL 圈" 是错的——Twitter 不是中文创作者聚集地。Phase 1 GTM 工时应明确分配：创始人小红书账号起号（**纳入 Phase 1 工时表**）。

2. **landing page 主标语必须是「把别人的爆款变成你的版本」**（§2.2 + §6.1），不是"AI 视频创作平台"。"AI 视频创作"在用户心智里已经被即梦/可灵/剪映占满，Cascade 在这个赛道**没有差异化**。**Phase 1 P1-1（首页）的文案应被产品设计师视为最高优先级**。

3. **Phase 1 Gate "至少 2 人付费"应升级为"至少 2 人预付 ¥39 且 30 天内使用 ≥ 3 次"**（§4.3）。当前 PHASED_PLAN Phase 1 Gate 没有付费验证（合理），但 Phase 2 Gate 的"5 人付 ¥39"是**容易被刷数据制造的幻觉**——必须搭配重复使用约束。建议 Phase 1 加一个"5 位用户表达预付意愿（含信用卡)"的软指标。

4. **赛道改写质量是真正的护城河，不是 URL 解析**（§6.2 + §3.2）。当前 PHASED_PLAN P1-3 工时给了 3 天——**这给少了**。建议 P1-3 改为 5 天，并且**Phase 1 用户必须集中在 2-3 个赛道**（宝妈辅食/育儿/家庭厨房）以便针对性优化 prompt。"赛道泛化"是 Phase 3 之后的事。

5. **Phase 1 不要让"锚点"成为用户感知到的主卖点**（§6.3）。当前 PRODUCT_VISION 把"锚点为主角"作为**最大差异化**——这是工程视角的差异化，**不是用户视角的卖点**。Phase 1 应让锚点**隐式自然涌现**（第二次创作时系统主动建议复用），不是 day-1 教育用户"什么是锚点"。等 Phase 2 数据证明 H4 后，再把锚点提升为显式卖点。
